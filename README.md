# PostScript — The editor LinkedIn forgot

> Write better LinkedIn posts. See exactly how they’ll land. Fix them before you publish.

**[→ Try it live](https://tadisweb.github.io/PostScript/)**

---

## The problem

LinkedIn’s editor does the basics.

But it leaves too much to guesswork:

- You can’t properly preview your post  
- You don’t know where the “see more” cut happens  
- Formatting is trial and error  
- There’s no feedback on whether what you’ve written actually works  

So most posts are written blind.

---

## The idea

PostScript gives you the writing environment LinkedIn never built.

A place where you can:

- Write with proper control  
- See exactly how your post will appear  
- Get real-time feedback as you go  
- Refine before anyone else sees it  

---

## What you get

- **Live LinkedIn preview**  
  Desktop + mobile views, including the “see more” fold  

- **Real-time stats**  
  Character count, true Unicode count, hook visibility, word count  

- **Post grader**  
  Clear signals on what’s working — and what isn’t  

- **Optional AI rewrites**  
  Generate alternative versions using your own API key (OpenAI / Anthropic)  

- **Templates**  
  Structured starting points when you don’t want to start from scratch  

- **Drafts + export/import**  
  Save locally, move between devices, never lose your work  

- **One-click copy**  
  Paste directly into LinkedIn exactly as it appears  

---

## Why it’s different

- Single HTML file  
- No sign-up  
- No installs  
- No tracking  

Everything runs locally in your browser.

Your writing stays with you.

---

## How to use it

1. Open the link  
2. Write your post  
3. Watch the preview and feedback update in real time  
4. Refine  
5. Copy → paste into LinkedIn  

---

## One line summary

If you’ve ever posted on LinkedIn and thought *“that didn’t land how I expected”* — this is for that.
